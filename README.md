# ecommerce-sales-analysis-
E-commerce sales data analysis using python and pandas

# Project Overview
This project analyzes e-commerce sales data using python and pandas to understand business performance.

## Tools Used 
- Python
- Pandas
- Matplotlib
- Jupyter Notebook
- 
## Analysis Performed
- Total sales and profit calculation
- Top 5 best-selling products
- Category-wise profit analysis
- Data visulization using pie charts

## key Insights
- Identified most profitable product category
- Found loss-making categories
- Provided data-driven business insights
